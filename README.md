# Lanka-project
Student Project: coordination platform for volunteer teams and humanitarian requests

Стек: 
Backend: Java, Spring Boot, Maven
Frontend: React, Vite
Database: PostgreSQL/MySQL, база даних у Supebase

Структура проєкту:
/backend — серверна частина на Java.
/frontend — клієнтська частина на React.

Як запустити проект:
Попередні вимоги:
JDK 17+
Node.js та npm/yarn

Запуск Backend:
Перейдіть у папку /backend.
Виконайте ./mvnw spring-boot:run (або запустіть на виконання клас Main.java).
Запуск Frontend:
Перейдіть у папку /frontend.
npm install
npm run dev
